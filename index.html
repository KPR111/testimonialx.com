FAQ

Frequently asked questions

Everything you need to know about TestimonialX.

How does the testimonial exchange work?
Two founders agree to swap testimonials. TestimonialX auto-posts both to X simultaneously. Each exchange costs $0.50 per user and results in verified, public social proof tied to real tweets.
Do I need a Twitter developer account?
No. You simply sign in with your regular X account via OAuth 2.0. TestimonialX handles the API integration on your behalf — no developer credentials needed.
Will TestimonialX post anything without my approval?
Never. Every testimonial goes through your approval queue first. The auto-post-to-X feature is opt-in per project — you choose when and if it fires.
How do I embed testimonials on my website?
Each project gets a widget.js snippet. Paste it on any website — Webflow, Framer, WordPress, Shopify, or custom HTML. The widget loads asynchronously and auto-updates as you approve new testimonials.
What happens if someone deletes their tweet?
Approved testimonials are stored in TestimonialX's database. Deletion from X does not remove them from your widget or public testimonial page.
Is there a subscription fee?
No subscriptions. TestimonialX uses a pay-as-you-go credit system. You start with $5 free credits on signup and buy more when you need them — no recurring charges.
Can I manage testimonials for multiple products?
Yes. Create separate projects for each site or product. Each project gets its own embed widget, testimonial wall, and public SEO page. One account handles them all.
